# Cubbard Marlboro Pics — Asset Repository

This repository should store the photo and field-reference material that supports the parametric model in `yazilimlar/cubbard-road-marlboro`.

## Recommended folder structure

```text
photos/
  00_original_uploads/
  01_site_plan_prints/
  02_route_9w_frontage/
  03_cubbard_drive_rear/
  04_drainage_wetland_low_points/
  05_septic_well_utilities/
  06_equipment_turning_reference/
manifest/
  site-photo-manifest.csv
```

## Naming convention

Use:

```text
YYYYMMDD_location_view_direction_subject_sequence.ext
```

Example:

```text
20260619_route9w_looking-north_entry-throat_001.jpg
```

## Minimum shot list

1. Route 9W frontage, both travel directions.
2. Proposed driveway/entry throat location.
3. Cubbard Drive / Old Indian Road rear relationship.
4. Low points, wet areas, swales, culverts, ditches, and visible drainage paths.
5. Existing house/barn/septic/well areas.
6. Neighboring property edges and screening/buffer areas.
7. Any wetland vegetation, ponding, stream, ditch, or waterbody evidence.
8. Reference photos of 70-ft to 100-ft equipment/lowboy turning needs.

## Field-photo metadata

Fill out `manifest/site-photo-manifest.csv` for every useful photo.
